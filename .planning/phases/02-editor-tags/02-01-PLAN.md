**Step 2: Create `src/stores/entryStore.ts`.**

Define the full entry store following the existing `uiStore.ts` Zustand `create()` pattern. The store must include:

```typescript
import { create } from 'zustand'
import { getDb } from '../lib/db'

interface Entry {
  id: string
  content: string
  mood: string | null
  word_count: number
  char_count: number
  created_at: number
  updated_at: number
  metadata: string
}

interface EntryState {
  entries: Entry[]
  selectedEntryId: string | null
  isSaving: boolean
  lastSavedAt: number | null

  loadEntries: () => Promise<void>
  selectEntry: (id: string) => Promise<void>
  createEntry: () => Promise<string>
  deleteEntry: (id: string) => Promise<void>
  saveContent: (entryId: string, content: string, wordCount: number, charCount: number) => Promise<void>
  updateMood: (entryId: string, mood: string | null) => Promise<void>
  updateCreatedAt: (entryId: string, timestamp: number) => Promise<void>
  ensureFirstEntry: () => Promise<void>
}
```

Key implementation details:
- `loadEntries`: `SELECT id, content, mood, word_count, char_count, created_at, updated_at, metadata FROM entries ORDER BY created_at DESC`
- `createEntry`: `INSERT INTO entries (content, word_count, char_count) VALUES ('', 0, 0)`. Then `SELECT id FROM entries ORDER BY created_at DESC LIMIT 1` to get the UUID. Call `loadEntries()` after. Set `selectedEntryId` to the new ID. Return the new ID.
- `selectEntry`: Set `selectedEntryId` to the given ID. (Auto-save flush will be added in Plan 02.)
- `deleteEntry`: `DELETE FROM entries WHERE id = ?`. Call `loadEntries()` after. If the deleted entry was selected, select the first remaining entry or set `selectedEntryId` to null.
- `saveContent`: `UPDATE entries SET content = ?, word_count = ?, char_count = ?, updated_at = ? WHERE id = ?` with `Date.now()` for updated_at. Set `isSaving = true` before, `isSaving = false` and `lastSavedAt = Date.now()` after. Also update the entry in the local `entries` array to keep sidebar word_count in sync without a full reload.
- `updateMood`: `UPDATE entries SET mood = ?, updated_at = ? WHERE id = ?`. Update local entries array.
- `updateCreatedAt`: `UPDATE entries SET created_at = ?, updated_at = ? WHERE id = ?`. Call `loadEntries()` to re-sort.
- `ensureFirstEntry`: If `entries.length === 0` after `loadEntries()`, call `createEntry()` to auto-create a blank entry per D-05.

**Step 3: Create `src/stores/tagStore.ts`.**

```typescript
import { create } from 'zustand'
import { getDb } from '../lib/db'

const TAG_COLORS = [
  '#EF4444', '#F97316', '#EAB308', '#22C55E',
  '#3B82F6', '#8B5CF6', '#EC4899', '#6B7280',
] as const

interface Tag {
  id: string
  name: string
  color: string
  created_at: number
}

interface TagWithCount extends Tag {
  usage_count: number
}

interface TagState {
  tags: TagWithCount[]
  loadTags: () => Promise<void>
  createTag: (name: string) => Promise<Tag>
  deleteTag: (id: string) => Promise<void>
  updateTagColor: (id: string, color: string) => Promise<void>
  addTagToEntry: (entryId: string, tagId: string) => Promise<void>
  removeTagFromEntry: (entryId: string, tagId: string) => Promise<void>
  getEntryTags: (entryId: string) => Promise<TagWithCount[]>
}
```

Key implementation details:
- `loadTags`: `SELECT t.id, t.name, t.color, t.created_at, COUNT(et.entry_id) AS usage_count FROM tags t LEFT JOIN entry_tags et ON et.tag_id = t.id GROUP BY t.id ORDER BY t.name COLLATE NOCASE ASC`
- `createTag`: Assign color using `TAG_COLORS[get().tags.length % TAG_COLORS.length]` per D-09. `INSERT INTO tags (name, color) VALUES (?, ?)`. Call `loadTags()` after. Return the created tag by selecting it: `SELECT * FROM tags WHERE name = ? COLLATE NOCASE`.
- `deleteTag`: `DELETE FROM tags WHERE id = ? AND id NOT IN (SELECT tag_id FROM entry_tags)` per Pitfall 4 safety. Call `loadTags()`.
- `updateTagColor`: `UPDATE tags SET color = ? WHERE id = ?`. Update local tags array.
- `addTagToEntry`: `INSERT OR IGNORE INTO entry_tags (entry_id, tag_id) VALUES (?, ?)`. Call `loadTags()` to refresh counts.
- `removeTagFromEntry`: `DELETE FROM entry_tags WHERE entry_id = ? AND tag_id = ?`. Call `loadTags()`.
- `getEntryTags`: `SELECT t.id, t.name, t.color, t.created_at, COUNT(et2.entry_id) AS usage_count FROM entry_tags et JOIN tags t ON t.id = et.tag_id LEFT JOIN entry_tags et2 ON et2.tag_id = t.id WHERE et.entry_id = ? GROUP BY t.id ORDER BY t.name COLLATE NOCASE ASC`

Export `TAG_COLORS` as a named export for use by the color picker component in Plan 03.
