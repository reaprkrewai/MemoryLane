**Scope:** SEC-01, SEC-02, SEC-03  
**Requirements:** User can set a PIN on first launch, change it anytime, and app auto-locks after idle timeout  
**Goal:** Secure app with numeric PIN-based lock and configurable auto-lock on inactivity

### 1.4: Auto-Lock on Idle Timeout

**File:** `src/hooks/useIdleTimeout.ts` (new), `src/stores/uiStore.ts` (extend)  
**Integration:** `App.tsx`

Track user activity (keydown, mouse move, click) and auto-lock after idle period:

```typescript
// In uiStore
create<UIState>((set, get) => ({
  idleTimeout: 5, // minutes, persisted
  lastActivityTime: Date.now(),
  isLocked: false,
  setIdleTimeout: (timeout: number) => set({ idleTimeout: timeout }),
  setLastActivityTime: () => set({ lastActivityTime: Date.now() }),
  setIsLocked: (locked: boolean) => set({ isLocked: locked }),
}))

// In useIdleTimeout hook
const useIdleTimeout = () => {
  const { idleTimeout, lastActivityTime, isLocked, setIsLocked, setLastActivityTime } = useUiStore();

  useEffect(() => {
    if (idleTimeout === 'never' || isLocked) return;

    const idleTimeMs = idleTimeout * 60 * 1000;
    const timer = setInterval(() => {
      const timeSinceActivity = Date.now() - lastActivityTime;
      if (timeSinceActivity > idleTimeMs) {
        setIsLocked(true);
      }
    }, 1000); // Check every second

    return () => clearInterval(timer);
  }, [idleTimeout, lastActivityTime, isLocked, setIsLocked]);

  // Track activity
  useEffect(() => {
    const handleActivity = () => setLastActivityTime();
    window.addEventListener('keydown', handleActivity);
    window.addEventListener('mousemove', handleActivity);
    window.addEventListener('click', handleActivity);

    return () => {
      window.removeEventListener('keydown', handleActivity);
      window.removeEventListener('mousemove', handleActivity);
      window.removeEventListener('click', handleActivity);
    };
  }, [setLastActivityTime]);
};
```

**Acceptance Criteria:**
- [ ] Idle timeout is configurable (1/5/15/30 min / never)
- [ ] Setting persists in `uiStore`
- [ ] Activity tracking works (keydown, mouse, click)
- [ ] App locks after idle period
- [ ] PIN screen shows when locked
- [ ] Unlocking app resets activity timer

## Implementation Notes

- **argon2-browser:** Use `hash()` function for PIN hashing. Provide `password`, `salt` (random bytes), and standard Argon2id params.
- **Activity Tracking:** Attach listeners at App.tsx level to catch all user interactions before they bubble down.
- **PIN Format:** 4-6 numeric digits only (no symbols). Numeric keypad UI makes this clear.
- **Error Messages:** Keep them simple and user-friendly. No security-through-obscurity (don't hide when the app is locked from attackers).
- **Deferred:** "Forgot PIN" recovery, biometric unlock, and account recovery are out of scope for v1.
