### 3.1: Extend UI Store with Settings State

**File:** `src/stores/uiStore.ts`

Extend the existing `uiStore` to persist theme, font size, and idle timeout:

```typescript
interface UIState {
  // Existing
  activeView: ActiveView;
  // New
  theme: 'light' | 'dark';
  fontSize: 'small' | 'medium' | 'large';
  idleTimeout: 1 | 5 | 15 | 30 | 'never'; // (from Plan 5.1)
  lastActivityTime: number; // (from Plan 5.1)
  isLocked: boolean; // (from Plan 5.1)

  // Setters
  setTheme: (theme: 'light' | 'dark') => void;
  setFontSize: (size: 'small' | 'medium' | 'large') => void;
  setIdleTimeout: (timeout: 1 | 5 | 15 | 30 | 'never') => void;
  setLastActivityTime: () => void;
  setIsLocked: (locked: boolean) => void;
}

const create<UIState>((set, get) => ({
  activeView: 'journal',
  theme: localStorage.getItem('theme') ?? 'dark',
  fontSize: localStorage.getItem('fontSize') ?? 'medium',
  idleTimeout: localStorage.getItem('idleTimeout') ?? 5,
  lastActivityTime: Date.now(),
  isLocked: false,

  setTheme: (theme) => {
    set({ theme });
    localStorage.setItem('theme', theme);
    applyTheme(theme); // Update DOM
  },
  setFontSize: (size) => {
    set({ fontSize: size });
    localStorage.setItem('fontSize', size);
    applyFontScale(size); // Update CSS custom property
  },
  setIdleTimeout: (timeout) => {
    set({ idleTimeout: timeout });
    localStorage.setItem('idleTimeout', String(timeout));
  },
  setLastActivityTime: () => set({ lastActivityTime: Date.now() }),
  setIsLocked: (locked) => set({ isLocked: locked }),
}))
```

**Acceptance Criteria:**
- [ ] `uiStore` includes all new fields
- [ ] Settings persist to localStorage
- [ ] Getters/setters are functional

```css
  /* ... existing dark tokens */

  --color-text: #1a1a1a;
  /* ... existing light tokens */
}
```

In `App.tsx`, apply theme and font scale on mount and when settings change:

```typescript
useEffect(() => {
  const { theme, fontSize } = useUiStore();
  
  // Apply theme
  if (theme === 'dark') {
    document.documentElement.classList.add('dark');
  } else {
    document.documentElement.classList.remove('dark');
  }
  
  // Apply font scale
  document.documentElement.classList.remove('font-small', 'font-medium', 'font-large');
  document.documentElement.classList.add(`font-${fontSize}`);
}, [theme, fontSize]);
```

## Implementation Notes

- **Persistence:** localStorage for theme/fontSize/idleTimeout. These settings don't need to sync across devices in v1.
- **Immediate Apply:** Changes take effect immediately without page reload.
- **Radio Buttons:** Use native `<input type="radio">` or shadcn/ui RadioGroup for consistency.
- **Font Scale:** CSS custom property `--font-scale` is set on `:root`. All font sizes should scale proportionally (use `rem` instead of `px` where possible).
- **Dark Mode:** Tailwind's dark mode (`.dark` class) already set up. Extend with custom color tokens if needed.
- **Sidebars:** Ensure Settings nav item doesn't break sidebar layout. Test with all nav items (Journal, Calendar, Search, Settings).
